# Earthquake Survival 🏠🏃‍♂️

A fast-paced, educational web-based game designed to simulate earthquake survival techniques. The goal is simple: find and stay under the safe zone (desk) before the timer runs out!

## 🎮 How to Play

1.  **Start:** Click the "START SIMULATION" button to begin.
2.  **Move:** Use the on-screen D-pad (mobile) or Keyboard (WASD/Arrow Keys) to navigate your character (the orange ball).
3.  **Survive:** Reach the highlighted **SAFE ZONE (DESK)** before the earthquake ends (when the timer reaches 0.0).
4.  **Avoid Obstacles:** You cannot move through furniture like cabinets and shelves. Navigate around them quickly!
5.  **Progress:** Each level increases the earthquake's intensity (magnitude) and reduces the time you have to find safety.

## ✨ Features

-   **Dynamic Shake Effect:** Visual feedback that intensifies as you level up.
-   **Collision System:** Realistic movement constraints with room furniture.
-   **Visual Particles:** Dust and debris effects during the quake for an immersive experience.
-   **Cross-Platform Controls:** Fully playable on both mobile touch devices and desktop browsers.
-   **Leaderboard:** Save your high scores and compete for the top spot (locally).
-   **Audio Feedback:** Sound effects for winning, losing, and the earthquake itself.

## 🛠️ Built With

-   HTML5 Canvas
-   Vanilla JavaScript
-   CSS3 Animations

## 🚀 How to Run

No installation or server required! Simply open `index.html` in any modern web browser.
